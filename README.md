# Tugas: Aplikasi To-Do List Interaktif

Deskripsi singkat:
Aplikasi To-Do List sederhana dibuat menggunakan HTML, CSS, dan JavaScript murni (tanpa framework). Fitur utama:
- Menambahkan tugas via form (validasi input tidak kosong).
- Menandai tugas sebagai selesai (tampilan dicoret).
- Menghapus tugas.
- Penyimpanan sederhana menggunakan localStorage (opsional, sebagai pengayaan).

## Cara menjalankan
1. Download atau clone repository.
2. Buka file `index.html` di browser modern (Chrome/Firefox/Edge).
3. Tambah, tandai, atau hapus tugas. Data akan disimpan di localStorage.

## Struktur file
- `index.html` — struktur HTML semantik.
- `style.css`  — stylesheet untuk tampilan.
- `script.js`  — logika aplikasi (DOM manipulation, event handling, DOM traversing).
- `README.md`  — dokumentasi singkat.

## Commit contoh (minimal beberapa commit yang menggambarkan proses)
- `git init`
- `git add .`
- `git commit -m "Add initial project files (html, css, js)"`
- `git commit -m "Add localStorage persistence and README"`

## Catatan untuk dosen
Aplikasi memenuhi ketentuan tugas: tidak menggunakan framework eksternal, menggunakan DOM API native, ada validasi input, dan menunjukkan DOM traversing saat handling event tombol.
